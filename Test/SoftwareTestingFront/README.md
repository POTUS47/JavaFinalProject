# SoftwareTestingFront

## 项目安装包及启动

```
npm install
```
```
npm run dev
```

## 开发期tips记录
1. **新增练习题**
    - **步骤**
        1. 在 src/assets/exercises/ 路径下创建新的练习数据json文件
        2. 在 src/assets/exercises/list.json 中添加新练习的信息
    - **注意点**
        - 练习的json格式中，输入"input"必须是数组格式
        - 代码格式：必须是完整的函数定义；函数名要与测试用例中调用的函数名一致
